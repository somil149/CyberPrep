# Changelog

All notable changes to CyberPrep Nexus will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-05-07

### Added
- Initial release of CyberPrep Nexus
- 14 career roles (5 Tier 1 priority + 9 Tier 2)
- Role-adaptive dashboard with personalized modules
- Question Bank with 35+ curated questions across all roles
- Mock Interview Simulator with timed sessions and self-scoring
- Threat Modeling Simulator (STRIDE, MITRE ATT&CK, DevSecOps Pipeline, Zero Trust)
- Agentic AI Design Studio (agent patterns, LLM security, AI red teaming, secure pipeline)
- Learning Roadmap with milestone tracking
- Flashcard system with mastery tracking
- Bookmark system for saving questions
- Offline-first PWA with IndexedDB persistence
- Dark cyber aesthetic design system
- GitHub Actions CI/CD with auto-deploy to GitHub Pages
